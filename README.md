# UEM Architect Website

A fusion website featuring structural elements inspired by reference sites and custom image materials, designed for an elevated visual experience.

## Overview

This project is the front-end structure for the UEM Architect Website. It brings together cohesive, branded aesthetics focusing on user experience, responsive layout, and interactive elements.

## Technologies Used

- **HTML5:** For structured visual presentation.
- **CSS3:** For responsive layout, theming, and aesthetics (Vanilla CSS).
- **JavaScript (ES6):** For functional logic, animations, and interactivity.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your favorite web browser.
3. No build tools are required for development.
