## 📋 Overview

This is the front-end codebase for the UEM Architect public website. It is a fully static site built with vanilla HTML, CSS, and JavaScript — no frameworks or build tools required.

### Page Sections

| Section | Description |
|---|---|
| **Hero** | Typing animation headline with primary CTAs |
| **Who We Serve** | Persona cards for IT Leaders, HR/Ops, and MSPs |
| **Services** | Six core service offerings with imagery |
| **Our Process** | Four-step timeline: Discover → Architect → Implement → Maintain |
| **Why UEM Architect** | Three differentiator cards |
| **Partner Logos** | Auto-scrolling trusted technology partner strip |
| **Testimonials** | Auto-advancing carousel with dot navigation |
| **CTA Banner** | Mid-page conversion prompt |
| **Contact** | Info panel + Formspree-powered contact form |
| **Footer** | Navigation, newsletter signup, and brand info |

---

## 🛠️ Technologies

- **HTML5** — Semantic markup with ARIA landmarks and accessibility attributes
- **CSS3** — Vanilla CSS with design tokens, responsive layout, and reduced motion support
- **JavaScript (ES6)** — Vanilla JS with IntersectionObserver, typing effect, carousel, and smooth scroll

## ⚙️ Configuration

Before deploying, update the following placeholders:

| File         | Placeholder                          | Replace With                          |
|---|---|---|
| index.html   | https://your-domain.com/             | Your live domain URL                  |
| index.html   | https://your-domain.com/og-image.jpg | Full URL to your OG image             |
| index.html   | YOUR_FORM_ID (contact form)          | Your Formspree form ID                |
| index.html   | YOUR_NEWSLETTER_ID (footer form)     | Your Formspree newsletter form ID     |
| index.html   | LinkedIn URLs                        | Your actual LinkedIn company page URL |
| index.css    | (none — all tokens are set)          | —                                     |
